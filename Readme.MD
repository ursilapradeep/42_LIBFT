# Libft

This project is a personal implementation of some standard C library functions. It is a foundational project to all beginners out there.. I have also attached test files..to check if the code was fine...

## 📁 Project Structure

Below is an overview of the key source files in `libft` and what each of them does:

### 🔠 Character Checking Functions
| Function        | Description                                |
|----------------|--------------------------------------------|
| `ft_isalpha.c`  | Checks if a character is alphabetic        |
| `ft_isdigit.c`  | Checks if a character is a digit           |
| `ft_isalnum.c`  | Checks if a character is alphanumeric      |
| `ft_isascii.c`  | Checks if a character is an ASCII character|
| `ft_isprint.c`  | Checks if a character is printable         |

---

### 🔤 Character Conversion Functions
| Function        | Description                                |
|----------------|--------------------------------------------|
| `ft_toupper.c`  | Converts lowercase to uppercase            |
| `ft_tolower.c`  | Converts uppercase to lowercase            |

---

### 🧵 String Functions
| Function         | Description                                    |
|------------------|------------------------------------------------|
| `ft_strlen.c`     | Returns the length of a string                |
| `ft_strchr.c`     | Locates a character in a string               |
| `ft_strrchr.c`    | Locates the last occurrence of a character    |
| `ft_strncmp.c`    | Compares two strings                          |
| `ft_strnstr.c`    | Locates a substring within a string           |
| `ft_strdup.c`     | Duplicates a string                           |
| `ft_strlcpy.c`    | Safe string copy                              |
| `ft_strlcat.c`    | Safe string concatenation                     |
| `ft_substr.c`     | Extracts a substring from a string            |
| `ft_strjoin.c`    | Concatenates two strings                      |
| `ft_strtrim.c`    | Trims characters from the beginning and end   |
| `ft_split.c`      | Splits a string into an array of strings      |

---

### 🔢 Memory Functions
| Function         | Description                                 |
|------------------|---------------------------------------------|
| `ft_memset.c`     | Fills memory with a constant byte          |
| `ft_memcpy.c`     | Copies memory area                         |
| `ft_memmove.c`    | Moves memory area                          |
| `ft_memcmp.c`     | Compares two memory areas                  |
| `ft_memchr.c`     | Searches memory for a character            |
| `ft_bzero.c`      | Zeroes out a memory area                   |
| `ft_calloc.c`     | Allocates memory and sets it to zero       |

---

### 🧮 Conversion Functions
| Function        | Description                                 |
|-----------------|---------------------------------------------|
| `ft_atoi.c`      | Converts a string to an integer             |
| `ft_itoa.c`      | Converts an integer to a string             |



### 🖨️ Output Functions (write to stdout)
| Function          | Description                                 |
|-------------------|---------------------------------------------|
| `ft_putchar_fd.c`  | Writes a character to a file descriptor    |
| `ft_putstr_fd.c`   | Writes a string to a file descriptor       |
| `ft_putendl_fd.c`  | Writes a string followed by newline        |
| `ft_putnbr_fd.c`   | Writes an integer to a file descriptor     |

---
